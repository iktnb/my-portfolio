# Personal Portfolio

A modern, dark-themed single-page portfolio for a Full-Stack Software Engineer. Built with React, Vite, and Tailwind CSS.

## Stack

- **React 18** + **TypeScript**
- **Vite** for dev and build
- **Tailwind CSS v4** with a custom theme (deep background, cyan/violet accents, glassmorphism cards)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview   # preview production build
```

## Customize content

- **Hero, About, Contact:** Edit `src/data/content.ts`
- **Tech stack:** Edit `src/data/tech.ts`
- **Projects:** Edit `src/data/projects.ts`

Update name, tagline, project titles/descriptions/links, and contact URLs in those files without touching component code.
